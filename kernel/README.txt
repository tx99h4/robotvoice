uImage -> noyau linux compressé utilisé par u-boot.

.config -> fichier de configuration des options du noyau, utilisez ce fichier en le copiant dans le répertoire décompressé des sources noyau avant d'effectuer la compilation, ce fichier est préconfiguré pour supporter Bluetooth.

System.map -> contient la liste des symboles utilisés par le noyau, fort utile lors du deboguage.
